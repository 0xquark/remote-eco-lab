The files

 - idle_analysis_script_Report.Rmd
 - idle_analysis_script.R
 - sus_analysis_script_Report.Rmd
 - sus_analysis_script.R

are defined specifically for either the SUS or Idle Mode measurement. If you need to change the title of the report, the names of the output files, etc. to be specific for the SUS or Idle Mode, these are the files that you will need to edit.

By contrast, the files

 - analysis_script_functions.R
 - analysis_script_Report.Rmd
 - analysis_script.R

are the same for BOTH SUS and Idle Mode measurements. If you edit these files, output for both measurements will be affected.
